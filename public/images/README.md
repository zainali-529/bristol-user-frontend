# Images Folder

## Required Images

### About Section
- **Filename**: `about-meeting.jpg`
- **Description**: Team meeting image showing people discussing energy solutions
- **Recommended Size**: 800x600px or similar 4:3 aspect ratio
- **Format**: JPG, PNG, or WebP

## How to Add Images

1. Place your image file in this folder: `public/images/`
2. Name it exactly as specified above
3. The image will automatically appear in the About section

## Image Optimization Tips

- Keep file size under 500KB for faster loading
- Use WebP format for better compression
- Ensure good lighting and professional quality
- Image should show a business meeting or team collaboration scene
